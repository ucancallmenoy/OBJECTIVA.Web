Single Inheritance

Single inheritance is a fundamental concept in object-oriented programming where a class (child or subclass) inherits properties and methods from another class (parent or superclass). This allows the child class to reuse and extend the functionality of the parent class.

  • Parent Class (Superclass): The class that provides its properties and behaviors to the child class.
  • Child Class (Subclass): The class that inherits and can modify or add new behaviors.

    • How does single inheritance works?

    In Java, a child class is created using the extends keyword to inherit from a parent class.

    For example, the following code snippet demonstrates a parent class called Animal and a child class called Dog that inherits from the Animal class.

    
public class Animal {{'{'}}
    public void eat() {{'{'}}
        System.out.println("Animal is eating");
    {{'}'}}
{{'}'}}

public class Dog extends Animal {{'{'}}
    public void bark() {{'{'}}
        System.out.println("Dog is barking");
    {{'}'}}
{{'}'}}

public class Main {{'{'}}
    public static void main(String[] args) {{'{'}}
        Dog dog = new Dog();
        dog.eat();  // Output: Animal is eating
        dog.bark(); // Output: Dog is barking
    {{'}'}}
{{'}'}}

    Parent and Child Classes

    The parent class provides base functionality while the child class inherits and can add or override behaviors.

    The super keyword

    It is used to refer to the parent class's methods or constructors

    For example, the following code snippet demonstrates how to use the super keyword to call the parent class's constructor.

    
class Animal {{'{'}}
    void sound() {{'{'}}
        System.out.println("Some generic sound");
    {{'}'}}
{{'}'}}

class Dog extends Animal {{'{'}}
    void sound() {{'{'}}
        super.sound(); // Calls the parent class's method
        System.out.println("Bark");
    {{'}'}}
{{'}'}}

public class Main {{'{'}}
    public static void main(String[] args) {{'{'}}
        Dog dog = new Dog();
        dog.sound();   // Output: Some generic sound Bark
    {{'}'}}
{{'}'}}

    Constructor Chaining

    It is a process of calling one constructor from another constructor with respect to the current object.

    Use super() explicitly to call a specific parent constructor.

    For example, the following code snippet demonstrates how to use constructor chaining in Java.

    Best Practices in Single Inheritance

    Use Proper "is-a" Relationships

    Ensure the child class is a specialized version of the parent class (e.g., a Dog is-a type of Animal).

    Minimize Overriding

    Only override parent class methods if absolutely necessary, to avoid confusion.

    Leverage 'super' Wisely

    Use 'super' to access parent class methods or constructors when required.

    Avoid Deep Hierarchies

    Keep the inheritance structure simple and avoid unnecessary levels of inheritance.
