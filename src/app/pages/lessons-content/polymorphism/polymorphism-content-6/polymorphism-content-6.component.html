Syntax for Overloading and Overriding

Overloading and overriding are two ways that methods in Java can be redefined in subclasses. Overloading is when a method has the same name but different parameters, while overriding is when a method has the same name and parameters as a method in its superclass.

Method overloading is like a Swiss Army knife with multiple tools (compile-time), while method overriding is like a custom-built tool that replaces a standard one for a specific job (runtime).

Method Overloading (Compile-time Polymorphism)

Method overloading happens when you define multiple methods with the same name but different parameters (number or type). The appropriate method is chosen by the compiler based on the method call.

Here's how to do it:


class Calculator {{'{'}}  
    int add(int a, int b) {{'{'}}  
        return a + b; // Adds two integers  
    {{'}'}}  

    int add(int a, int b, int c) {{'{'}}  
        return a + b + c; // Adds three integers  
    {{'}'}}  

    double add(double a, double b) {{'{'}}  
        return a + b; // Adds two double values  
    {{'}'}}  
{{'}'}}  

public class Main {{'{'}}  
    public static void main(String[] args) {{'{'}}  
        Calculator calc = new Calculator();  
        System.out.println("Sum of 2 integers: " + calc.add(3, 5)); // Output: 8  
        System.out.println("Sum of 3 integers: " + calc.add(1, 2, 3)); // Output: 6  
        System.out.println("Sum of 2 doubles: " + calc.add(2.5, 3.5)); // Output: 6.0  
    {{'}'}}  
{{'}'}}

Method Overriding (Runtime Polymorphism)

Method overriding occurs when a subclass provides its specific implementation of a method that is already defined in its parent class. The appropriate method is based on the actual object type, not the reference type.

Here's how to do it:


class Animal {{'{'}}  
    void sound() {{'{'}}  
        System.out.println("Some generic animal sound"); // Default sound  
    {{'}'}}  
{{'}'}}  

class Dog extends Animal {{'{'}}  
    {{'@Override'}}
    void sound() {{'{'}}  
        System.out.println("Bark"); // Dog-specific sound  
    {{'}'}}  
{{'}'}}  

class Cat extends Animal {{'{'}}  
    {{'@Override'}}
    void sound() {{'{'}}  
        System.out.println("Meow"); // Cat-specific sound  
    {{'}'}}  
{{'}'}}  

public class Main {{'{'}}  
    public static void main(String[] args) {{'{'}}  
        Animal myAnimal = new Animal();  
        Animal myDog = new Dog(); // Polymorphism: Superclass reference, subclass object  
        Animal myCat = new Cat();  

        myAnimal.sound(); // Output: Some generic animal sound  
        myDog.sound(); // Output: Bark  
        myCat.sound(); // Output: Meow  
    {{'}'}}  
{{'}'}}

But how to implement these two?

You are tasked to create a program for a Pet Care Center. The program needs to:

  • Calculate the food requirement for different pets based on their size or weight. (This uses method overloading because the calculation method needs different parameters.)
  • Allow pets like dogs and cats to make their own sounds when prompted. (This uses method overriding because the sound method is implemented differently for each pet type.)
    • 
// Parent class for all pets
class Pet {{'{'}}  
    // Method to calculate food (overloading example)
    int calculateFood(int weight) {{'{'}}  
        return weight * 2; /* Basic food calculation for weight */  
    {{'}'}}  

    int calculateFood(int weight, int age) {{'{'}}  
        return (weight * 2) + (age * 1); /* Includes age in the calculation */  
    {{'}'}}  

    // Method to make a sound (overridden in subclasses)
    void sound() {{'{'}}  
        System.out.println("Some generic pet sound"); /* Default sound */  
    {{'}'}}  
{{'}'}}  

// Subclass for Dog
class Dog extends Pet {{'{'}}  
    {{'@Override'}}  
    void sound() {{'{'}}  
        System.out.println("Bark"); /* Dog-specific sound */  
    {{'}'}}  
{{'}'}}  

// Subclass for Cat
class Cat extends Pet {{'{'}}  
    {{'@Override'}}  
    void sound() {{'{'}}  
        System.out.println("Meow"); /* Cat-specific sound */  
    {{'}'}}  
{{'}'}}  

public class Main {{'{'}}  
    public static void main(String[] args) {{'{'}}  
        // Method overloading example
        Pet genericPet = new Pet();  
        System.out.println("Food for pet (weight 10): " + genericPet.calculateFood(10));  
        /* Output: Food for pet (weight 10): 20 */  
        System.out.println("Food for pet (weight 10, age 2): " + genericPet.calculateFood(10, 2));  
        /* Output: Food for pet (weight 10, age 2): 22 */  

        // Method overriding example
        Pet myDog = new Dog();  
        Pet myCat = new Cat();  
        myDog.sound(); /* Output: Bark */  
        myCat.sound(); /* Output: Meow */  
    {{'}'}}  
{{'}'}}
